# Cross Country

Creating a componennt library is essential for re-using components and utility code across various projects.

Cross Country is ideal for crafting apps that are rich with personal metrics, encourage self reflection, and exploring new and old worlds. Personally, I am using these components to build a link curation app and a rogue-like game based on history following the Viking migration. The goal is a project that could generate revenue and promote creators.

It should be effortless to collect, curate, and create content as well as track its effectiveness across other social platforms and schedule activities.

This library is a collection of components that are geared towards building a community around content curation, courses and tile-based games. There's great potential in discovering a new product by combining elements of all of them!
