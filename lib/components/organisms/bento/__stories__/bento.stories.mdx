# Bento

The bento component provides a grid layout for displaying content and we can pass in components for each slot by the `boxes` prop. 

The grid can be configured to have a specific number of rows and columns. The border radius and the gap between the grid items can also be configured. 

There are 4 layouts that can be used: 2x2, 3x3, 4x4, 5x4.

Only the 2x2 layout has a special prop `isMainRight` to reverse the position of the main component placing it on either the right or left side of the grid.
