# One Acre

<Subtitle>The Dream to Work a One Acre Farm for a Day</Subtitle>

<Paragraph>Can I convince my daughter that tile games are anywhere near as fun as Roblox?!</Paragraph>

<RelatedArticles articles={[{id: 0, title: 'Investigation: How Roblox Is Exploiting Young Game Developers', url: 'https://www.youtube.com/watch?v=_gXlauRB1EQ'}]} />

<Paragraph>
Inspired by an epside of Lost when Desmond mentions the <Link url="https://lostpedia.fandom.com/wiki/Snowglobe">Snowglobe</Link> and Brad hypes <Link url="https://mythic-quest.fandom.com/wiki/Battle_Royale_mode">Battle Royale Mode</Link> on Mythic Quest, I wanted to build a small, diorama-type experiment which features an Area of Affect where a circle will the affect tiles hilighting those within the circle and darkening those beyond. 
</Paragraph>
<Paragraph>
As the cirlce shrinks, the agents will need to move closer to the center and come in closer contact. Each agent will be able to cast an AOE spell that could push (a nod to <Link url="https://www.polygon.com/2018/2/26/17053646/into-the-breach-preview-impressions-difficulty-ftl-roguelike-pc">Into the Breach</Link>) the other agent outside of the circle and perish.
</Paragraph>
